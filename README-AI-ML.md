## Preparation Tips by Track

### LLM/ML Research (OpenAI, Anthropic, DeepMind, FAIR)
- Strong ML/AI coursework + published papers preferred
- PyTorch proficiency, transformer architectures
- A single impressive project beats many mediocre ones
- Apply early — these fill fast

### Computer Vision / Generative AI (Runway, Midjourney, NVIDIA DGX)
- Portfolio of generative models or diffusion model work
- Experience with stable diffusion, GANs, or video generation
- PyTorch + CUDA knowledge

### NLP (Apple Siri, Converse AI, Cohere)
- NLP coursework, Hugging Face experience
- LLM fine-tuning or RAG pipeline experience
- HF Hub profile with model/dataset uploads helps

### Robotics / Embodied AI (Figure AI, Tesla)
- RL, control theory, perception networks
- ROS or simulation experience (Gazebo, MuJoCo)
- Real-world robotics projects
