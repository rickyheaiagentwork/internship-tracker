## What You'd Actually Do

### Big Tech SWE Interns (#1–6, 8–9, 14–16, 20)
- Write production code for real products (Search, Maps, AWS, Office, CRM, etc.)
- 12-week summer program with mentor assigned from day one
- OA (coding assessment) → technical interviews → offer
- Interns ship real features used by millions

### Cloud / Platform AI (#35–37, 56)
- Build the infrastructure layers: SageMaker, Azure OpenAI, Watsonx, Vertex AI
- Deploy ML models at enterprise scale
- Work with large-scale distributed systems (Spark, Kubernetes, cloud-native)

### ML Infrastructure / Silicon (#44–45)
- Cerebras: AI chip design, compiler tooling, model training at unprecedented scale
- DeepMind Research: publishable research on RL, language models, or neuroscience-inspired computing

### Data Science / Analytics (#10, 51)

### Fellowships & Scholarships (#11, 12, 13, 17, 19)

- MLH Fellowship ($5K/term): OSS contributions to Mozilla, Linux Foundation, etc.
- Amazon Future Engineer: underrepresented students in computing

## Preparation Tips

### For SWE Internships (Big Tech)
- Practice LeetCode Medium/Hard daily for OA rounds
- Build a polished GitHub with 2-3 substantial projects
- Know your resume projects inside-out
- Most OAs are HackerRank or CoderPad-style coding tests

### For Data / Platform Roles
- Strong SQL + Python + one cloud platform (AWS/GCP/Azure)
- Familiarity with Docker, Kubernetes, CI/CD pipelines
- Spark/MLOps experience is a big plus

### For Fellowships
- Research background + publications preferred (Google PhD: required)
- Write strong essays explaining your motivation and impact goals
- Letters of recommendation 2-3 weeks before deadline
