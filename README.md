## 🛠️ How to Use This Tracker

### In the App (index.html)
1. **Open `index.html`** in your browser
2. **Browse cards** — filter by season, type, or status
3. **Click any card** → see full details + "Apply Now" button
4. **Update status** as you progress: Discovered → Applied → Interviewing → Offer/Accepted/Rejected
5. **Track your resume projects** per entry under "Resume Alignment"
6. **Add new targets** with the "+ Add Target" button

### Managing Your Data
- **Export:** Click "Export JSON" to download a backup of all your data
- **Import:** Click "Import JSON" to load a previously exported file
- **Cross-device:** Export from one device, import on another

### Git Sync
All data lives in `data/tracker.json`. Commit and push regularly:
```bash
git add -A && git commit -m "Updated tracker: applied to X, got interview at Y"
git push
```
